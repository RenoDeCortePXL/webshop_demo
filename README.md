# Webshop Demo
    Vue.js, Vuetify, Vuex, Node.js, VueSax, Express.js & MySQL

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

## Run Backend
(second terminal / cmd)
```
cd project
cd server
npm install
cd src
nodemon app.js
```
